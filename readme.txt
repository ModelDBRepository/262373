This is the readme for the models associated with the paper:

Coggan JS, Keller D, Calì C, Lehväslaiho H, Markram H, Schürmann F, Magistretti PJ (2018) Norepinephrine stimulates glycogenolysis in astrocytes to fuel neurons with lactate. PLoS Comput Biol 14:e1006392

This code was contributed by JS Coggan.

To run compile the mod file and start the hoc program with NEURON.
It will take a few hours to run and generate the data files.
If you need more help on your platform please consult this web page:
https://senselab.med.yale.edu/ModelDB/NEURON_DwnldGuide
